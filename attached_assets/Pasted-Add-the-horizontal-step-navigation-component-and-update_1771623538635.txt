Add the horizontal step navigation component and update existing pages to use it.

━━━ STEP 1: STEP NAV COMPONENT ━━━

Create file: components/StepNav.tsx

Props:
  steps: string[]          — e.g. ['Lookup', 'LC Check', 'TwinLog Trail', 'Archive']
  currentIndex: number     — 0-based index of active step
  completedUpTo: number    — steps before this index show as done (green check)

Render: horizontal bar, background var(--s1), border-bottom 1px solid var(--s5), padding 0 24px

Each step:
  Container: flex row, align-items center, padding 13px 14px 11px, position relative
  Active state: border-bottom 2px solid var(--blue), margin-bottom -1px
  
  Step dot (18×18, border-radius 50%):
    Done  → background var(--green), border var(--green), shows "✓" in white 9px
    Active → background var(--blue),  border var(--blue),  shows step number in white 9px
    Pending → border 1.5px solid var(--t3), shows step number in var(--t3) 9px

  Step label (12px, font-weight 600):
    Done → color var(--t2)
    Active → color var(--t1)
    Pending → color var(--t3)

  Arrow separator "›" between steps: color var(--s5) 12px, margin 0 2px
