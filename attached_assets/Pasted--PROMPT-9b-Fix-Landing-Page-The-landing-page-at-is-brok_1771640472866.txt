### Step 2 â€” Ensure fonts load globally

```html
<link href="https://fonts.googleapis.com/css2?family=Fraunces:wght@700;900&family=Plus+Jakarta+Sans:wght@400;500;600;700&family=DM+Mono:wght@400;500&display=swap" rel="stylesheet"/>
```

### Step 3 â€” Ensure global background is set

In `globals.css`, make sure:

```css
html, body {
  background: #0D1117;
  margin: 0;
  padding: 0;
}
```

This prevents any white flash or wrong background showing through.

### Step 4 â€” Do NOT apply sidebar layout to the homepage

The sidebar layout (with left nav) should only wrap these routes:
- `/lookup`
- `/lc-check`
- `/dashboard`
- `/trades`
- `/templates`
- `/alerts`
- `/demurrage`
- `/settings`
- `/admin/*`

The homepage `/` and `/pricing` should be full-width standalone pages with the top nav only.

If you are using a root layout that applies the sidebar to all pages, move the sidebar into a nested layout that only covers the app routes above. The homepage must be excluded.
